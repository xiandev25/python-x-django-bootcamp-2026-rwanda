# Python x Django Bootcamp 2026 Rwanda 🇷🇼🐍

This repository is for recording my work and experience throughout the **Python x Django Bootcamp 2026 Rwanda**, co-organized by [Python Rwanda](https://www.linkedin.com/company/python-rwanda/) and [Django Rwanda](https://www.linkedin.com/company/django-rwanda/).

This bootcamp is an 8-week program starting from February 16 and ending on April 16, 2026. The topics covered are distributed as follows:
- Week 1: **Python Fundamentals**
- Week 2: **Django MVT Architecture**
- Week 3: **PostgreSQL & Database Design**
- Week 4: **API Development (DRF)**
- Weeks 5-8: **A Final Project**
